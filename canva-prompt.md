# Canva Prompt: Light Minimal Portfolio Website Design

## Design Brief

Create a modern, light, and minimal portfolio website design inspired by professional developer portfolios. The design should emphasize clean aesthetics, ample white space, and elegant typography.

## Color Palette
- **Primary Background**: Pure white (#FFFFFF) or soft off-white (#FAFAFA)
- **Secondary Background**: Light gray (#F5F5F5) for section dividers
- **Text Color**: Dark charcoal gray (#2C2C2C) or deep slate (#1A1A1A) for primary text
- **Accent Color**: Soft, muted tones - consider:
  - Light blue-gray (#E8F0F6)
  - Soft lavender (#F5F3FF)
  - Warm beige (#F9F7F4)
  - Or a single subtle accent color of your choice (avoid bright/vibrant colors)
- **Border/Divider**: Very light gray (#E5E5E5)
- **Hover States**: Subtle gray tint (#F0F0F0)

## Typography
- **Headings**: Modern sans-serif font (e.g., Inter, Poppins, or Montserrat) - Bold weight
- **Body Text**: Clean sans-serif font (same family as headings) - Regular/Medium weight
- **Font Sizes**: 
  - Hero heading: 48-64px
  - Section headings: 32-40px
  - Body text: 16-18px
  - Small text: 14px
- **Line Height**: Generous (1.6-1.8 for body text)
- **Letter Spacing**: Slightly increased for headings (0.5-1px)

## Layout Structure

### Header/Navigation
- Fixed or sticky navigation bar at the top
- White/transparent background with subtle shadow on scroll
- Logo/name on the left
- Navigation menu on the right: Home, About, Projects, Skills, Contact
- Minimal underline or dot indicator for active page
- Smooth hover transitions

### Hero Section
- Full-width section with generous padding (80-120px vertical)
- Centered content alignment
- Large, bold heading introducing the name/title
- Short tagline or subtitle (1-2 lines max)
- Optional: Subtle geometric shapes or minimal illustrations in the background (very light, barely visible)
- Call-to-action button (if needed) with soft rounded corners and minimal border

### About Section
- Clean two-column layout (text on left, image/illustration on right, or vice versa)
- Or single column with centered content
- Generous spacing between paragraphs
- Optional: Soft background color change (very subtle)

### Projects/Portfolio Section
- Grid layout (2-3 columns on desktop, 1 column on mobile)
- Project cards with:
  - White background
  - Subtle shadow or border
  - Project thumbnail/image
  - Project title
  - Brief description (2-3 lines)
  - Tags/technologies used (small, minimal badges)
- Hover effect: Slight elevation or subtle color shift
- Generous spacing between cards

### Skills Section
- Clean list or grid of skills
- Minimal icons or simple text labels
- Grouped by category (Frontend, Backend, Tools, etc.)
- Soft background or border around each skill item

### Contact Section
- Centered content
- Simple contact form or contact information
- Social media icons (minimal, monochrome style)
- Generous white space around elements

### Footer
- Minimal footer with:
  - Copyright text
  - Social links (if not in contact section)
  - Optional: Back to top button
- Light gray background or border-top separator

## Design Principles

1. **White Space**: Use generous padding and margins throughout (minimum 40-60px between sections)
2. **Minimalism**: Remove all unnecessary elements - every element should have a purpose
3. **Consistency**: Maintain consistent spacing, typography, and color usage throughout
4. **Subtle Interactions**: Use gentle hover effects, smooth transitions (200-300ms)
5. **Clean Lines**: Use subtle borders, dividers, or shadows instead of heavy visual elements
6. **Grid System**: Align all elements to a consistent grid (12-column or 16-column)
7. **Visual Hierarchy**: Use size, weight, and spacing to guide the eye naturally

## Visual Elements

- **Shadows**: Very subtle, soft shadows (avoid harsh drop shadows)
- **Borders**: Thin (1px) light gray borders where needed
- **Icons**: Minimal, line-style icons (avoid filled/bold icons)
- **Images**: High-quality, well-lit images with consistent style
- **Shapes**: If using geometric shapes, keep them very subtle and minimal
- **Animations**: None or very subtle fade-in on scroll (if Canva supports)

## Responsive Considerations

- Mobile-first approach
- Stack columns vertically on smaller screens
- Reduce font sizes appropriately (but maintain readability)
- Touch-friendly button sizes (minimum 44x44px)
- Maintain generous spacing even on mobile

## Content Sections to Include

1. Navigation/Header
2. Hero/Introduction
3. About Me
4. Skills/Technologies
5. Projects/Portfolio
6. Experience (optional)
7. Contact
8. Footer

## Final Notes

- The overall feel should be: **Clean, Professional, Modern, Approachable**
- Avoid: Dark colors, heavy shadows, busy patterns, excessive animations, cluttered layouts
- Focus on: Content readability, visual breathing room, elegant simplicity
- Think: Apple website aesthetic, but for a personal portfolio
